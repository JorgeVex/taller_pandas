# Taller de Pandas: DATASET Impacto de la IA en Estudiantes

Entregable del Bootcamp en Data Science - Escuela BIT

## Descripción

Análisis exploratorio con Pandas sobre un dataset de 50,000 estudiantes que evalúa el uso de herramientas de IA generativa y su relación con el rendimiento académico, la ansiedad y el riesgo de burnout.

## Preguntas de análisis

1. ¿El uso intensivo de IA generativa está asociado con cambios en el GPA?
2. ¿Existe relación entre la dependencia percibida de la IA y el burnout/ansiedad?
3. ¿Cómo varía el uso de IA y su impacto según la carrera y la política institucional?

## Contenido del repositorio

- `taller_pandas_ia_estudiantes.ipynb`: notebook con la carga, limpieza, análisis y conclusiones.
- `ai_student_impact_dataset__1_.csv`: dataset utilizado.

## Herramientas

- Python
- Pandas
- Matplotlib

## Cómo ejecutar

1. Clonar el repositorio.
2. Asegurarse de tener instaladas las librerías: `pandas`, `numpy`, `matplotlib`.
3. Abrir `taller_pandas_ia_estudiantes.ipynb` en Jupyter Notebook y ejecutar todas las celdas.
